# ChatWebX

## Project Overview
ChatWebX is a versatile chat application that allows users to engage in real-time conversations. Designed for ease of use and high performance, it aims to facilitate seamless communication across various platforms.

## Features
- Real-time messaging
- User authentication
- Multi-device support
- Emoji and file sharing
- Group chat functionality

## Tech Stack
- **Frontend:** React.js, Redux
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **WebSocket:** Socket.IO
- **Deployment:** Docker, Heroku

## Installation
To set up the project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/rajofearth/chatwebx.git
   ```
2. Navigate to the project directory:
   ```bash
   cd chatwebx
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage
After setting up the application, open your browser and navigate to `http://localhost:3000` to access ChatWebX. Sign up or log in to start chatting!
